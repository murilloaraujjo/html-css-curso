Listas

Listas Ordenadas (Ordered Lists)

Usa-se a tag <ol> para delimitar a lista e a tag <li> para identificar cada item da lista:

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas (Unordered Lists)

São listas que usam marcadores sem idicar a ordem da lista

Usa-se a tag <ul> para delimitar a lista:

  • item 1
  • item 2
  • item 3

Listas Mistas

  1. item 1
  2. item 2
    • item 2.1
    • item 2.2
  3. item 3
  4. item 4

Listas de Definições (Definition lists)

Usa a tag <dl> para abrir a lista, a tag <dt> para o termo e a tag <dd>para descrição do termo. Exemplo:

HTML
Linguagem de marcação para criar conteúdos de sites.
CSS
Linguagem de marcação para especificação de estilos em sites
Javascript
Linguagem de programação utilizada para criar interatividades em sites
